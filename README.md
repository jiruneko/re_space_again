# Re:Space（リ・スペース）

> ここでは、何もしなくていい。

Re:Space は、「生きているだけで、そこにいていい」と感じられる  
**居場所（Ibasho）支援型のミニマル SNS / Web アプリ**です。

承認・競争・評価から一度距離を置き、  
ただ“存在する”ことを許される空間を、デジタル上に作ることを目的としています。

---

## 🌱 コンセプト

多くの SNS は、次のような前提を持っています。

- 何かを投稿しなければならない  
- 価値ある発言・成果・成長を示さなければならない  
- 数字（いいね・フォロワー）で評価される  

Re:Space は、**それらをすべて前提にしません。**

- 投稿しなくてもいい  
- 役に立たなくてもいい  
- 成長しなくてもいい  

**「ここにいるだけでいい場所」**  
それが Re:Space です。

---

## ✨ 現在の機能（MVP）

- 🏠 **居場所画面**
  - 静的で落ち着いた空間
  - 何も操作しなくても成立する UI
- 🎧 **環境音・BGM**
  - 集中や安心感を邪魔しない設計
- 🕊 **超ミニマル設計**
  - 通知・タイムライン・バズ要素なし
  - 「やらされる」体験を排除

※ あえて機能を削り、**“余白”そのものを価値**としています。

---

## 🧠 想定ユーザー

- 疲れている人
- SNS や社会的評価に消耗している人
- 何かを「生産」する気力がない人
- ただ安心できる場所が欲しい人
- メンタルヘルスに課題を感じている人

Re:Space は「治療」ではなく、  
**回復の手前にある“安全な空間”**を目指しています。
